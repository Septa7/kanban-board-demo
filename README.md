# kanban-board-demo
Demo-Projekt zum Erlernen von GitHub-Projekten, Issues und Kanban-Workflows
# Kanban Board Demo
Ein einfaches GitHub-Demoprojekt, das zeigt, wie man Aufgaben mit
*Issues*, *Projektboards*, *benutzerdefinierten Feldern* und *Automatisierungen* verwalten kann.

# Funktionen
- Erstellen und Kategorisieren von Issues (Labels, Bearbeiter, Meilensteine)
- Verwalten von Aufgaben in einem Kanban-ähnlichen Projektboard
- Verwenden von benutzerdefinierten Feldern (Priorität, Status, Aufwand)
- Automatisieren von Workflows (z. B. automatisches Schließen von Issues nach dem Mergen eines PR)
